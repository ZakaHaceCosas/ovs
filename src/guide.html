Troubleshooting

If you encounter any issues while using the program, try the following troubleshooting steps:

  1. Ensure these two files exist in the AppData/Roaming path of your user directory: data.json and prefs.json. Also make sure they both have content. The prefs.json should look similar to this: 
    2. {
    "theme":"dark",
    "lang":"english",
    "appName":"OVS 3",
    "startup":false,
    "encrypt":false
}
  2. Make sure that Windows is not interfering with how the program does access these two files.
  3. Try restarting the program, or reinstalling it if needed (your data should not be lost by reinstalling it).
